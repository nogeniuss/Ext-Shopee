# Ext-Shopee
Documentação Completa

script.js:

Descrição: Este script é responsável por adicionar um botão "Copiar" à página da Shopee e copiar o conteúdo de elementos específicos para a área de transferência.

manifest.json:

Descrição: Este arquivo define as configurações da extensão, como nome, versão, permissões e quais arquivos serão carregados.

style.css:

Descrição: Este arquivo define o estilo visual do botão "Copiar".

_o Codigo tem Id repetidos, e a div do button esta incorreta, o objetivo é entende o funcionamento de uma extensão e manipular dados_
